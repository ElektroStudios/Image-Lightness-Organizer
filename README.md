## 👋 Introduction

**Image Lightness Organizer** is designed to analyze and categorize local image collections. By computing the average lightness ($L^*$) of each image using the scientifically accurate **CIE L* color space**, the application eliminates manual sorting, grouping your media files into structured, percentage-based luminance brackets.

## 💡 Motivation

When working with large collections of graphic assets, datasets for machine learning, or wallpaper libraries, sorting images by their visual weight and luminance manually is nearly impossible. Standard file managers only allow sorting by name, date, or dimensions, leaving no native option to group files by how bright or dark they actually appear to the human eye. 

Furthermore, standard RGB math fails to reflect true human perception. I built this small command-line utility to solve this problem using the CIE L* color space, providing a lightweight, automated way to cluster image assets into precise percentage brackets without complex overhead.

##### ⚡ The Real Question
###### How much time do you lose manually hunting for light or dark backgrounds, assets, or textures across disorganized folders when a perceptually accurate algorithm can catalog them for you in seconds?.

## 🤖 Features

- Calculates average image lightness based on human visual perception using the CIE L* color space rather than simple RGB averages.
- Non-destructive processing that safely scans top-level files within the source directory only, ignoring subfolders.
- Creates a dedicated `@Sorted by lightness` output folder and automatically distributes image files into 10 distinct percentage-based subfolders:
  - Light 00%-10%
  - Light 10%-20%
  - Light 20%-30%
  - Light 30%-40%
  - Light 40%-50%
  - Light 50%-60%
  - Light 60%-70%
  - Light 70%-80%
  - Light 80%-90%
  - Light 90%-100%
- Supports the following image formats:
  - JPG / JPEG
  - TIFF / TIF
  - BMP
  - PNG
  - WEBP
  - ICO
  - AVIF
